## Usage

1. Déboguer un script avec des arguments :
   - Configurez `launch.json` (voir section précédente) puis lancez F5.
   - Une invite vous demandera les arguments, entrez-les séparés par des espaces.

2. Exécuter les tests unitaires :

```bash
python -m pip install pytest  # si nécessaire
pytest
```

Les tests couvrent la fonction `saluer` et l'appel en ligne de commande.
